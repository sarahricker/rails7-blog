# Rails7 Blog
From the [GoRails: Build a Blog with Rails 7](https://gorails.com/episodes/blog-rails-new?autoplay=1) Guide

## Ruby version

3.0.0

## System dependencies

- devise
- tailwindcss

## Local development

- `brew install postgresql`
- `brew services start postgresql`
- `bundle install`
- `bin/dev`
- - Starts web and css processes for tailwindcss
